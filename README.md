# ChoreoTrajMirrorer

To use this program, download `ChoreoTrajMirrorer.jar` and place it under whichever directory contains your Choreo `.traj` files. Double click the `.jar`, click `Select .traj Files`, select the `.traj` files you intend to mirror, and click `Mirror Selected`. The mirrored paths will output to the directory that contains the `.jar`.
